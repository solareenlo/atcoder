## [B - Picking Up](https://atcoder.jp/contests/diverta2019-2/tasks/diverta2019_2_b)

- もしくはループ2回で map を使う 全探索
- 問題文を読み替えると，
	- ２次元平面上に点が存在し，
	- 任意の2点間のベクトルと同じベクトルの個数をカウントしていき，
	- そうしてカウントした最大数を N から引いた数が答え．
