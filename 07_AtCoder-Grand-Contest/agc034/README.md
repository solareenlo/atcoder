## [A - Kenken Race](https://atcoder.jp/contests/agc034/tasks/agc034_a)
- `C > D` の時，
    - A が B を追い抜かないといけないので，
    - しかも，A も B も右にしか動けないので，
    - B と D の間に`.` が3つ必要になる．
- あとは，`#` が2連続あると移動することはできない．
