## [C - 4-adjacent](https://atcoder.jp/contests/abc069/tasks/arc080_a)
- 2が存在するかしないかで場合分けを行う．
- 2が存在する場合は，2を右端に持っていって，4と奇数の組みを1つずつ作れば良い．
    - `4奇4奇4奇422222`
- 2が存在しない場合は，両端に奇数を置いて，`奇4奇4奇4奇4奇` の様に置けば良い．
- パターン発見．
