# [AtCoder Beginner Contest 053](https://atcoder.jp/contests/abc053/tasks)

## [D - Card Eater](https://atcoder.jp/contests/abc053/tasks/arc068_b)
- 最初のカードの枚数は常に奇数なので，
- カードの山に `k` 種類のカードがあった場合，
	- `k` が奇数なら答えは，`k`．
	- `k` が偶数なら答えは，`k - 1`．
