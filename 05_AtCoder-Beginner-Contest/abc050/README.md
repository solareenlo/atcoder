# [AtCoder Beginner Contest 050](https://atcoder.jp/contests/abc050)

## [C - Lining Up](https://atcoder.jp/contests/abc050/tasks/arc066_a)
- パターン を見つける問題．
- N が奇数の時，
	- A を昇順に並び替えると，
	- `0, 2, 2, 4, 4, ...`
	- というふうに，0 では無い，N より小さい偶数が2個ずつ並んでいる．
	- ので，`0, 1, 2, 3, 4, 5, ...` の偶数番目の数に `+1` した数列と比較し，A があり得るかどうかの判定を行えば良い．
- N が偶数の時，
	- A を昇順に並び替えると，
	- `1, 1, 3, 3, 5, 5, ...`
	- というふうに，N より小さい奇遇が2個ずつ並んでいる．
	- ので，`0, 1, 2, 3, 4, 5, ...` の奇数番目の数に `+1` した数列と比較し，A があり得るかどうかの判定を行えば良い．
- A があり得た場合は，`pow(2, N / 2)` を計算したのが答え．
- `N == 1` の時の例外処理に気を付ける．
