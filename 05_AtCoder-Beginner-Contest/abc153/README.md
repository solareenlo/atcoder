## [E - Crested Ibis vs Monster](https://atcoder.jp/contests/abc153/tasks/abc153_e)
- 動的計画法
- dp[i] = モンスターの体力を i 減らすために消耗する魔力の最小値
- ここで，『同じ魔法は何度でも使うことができる』という設定になっているので，ループは1回で完結することができる．
